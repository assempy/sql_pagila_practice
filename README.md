# SQL Practice with Pagila Database

This repository contains SQL exercises completed using the **Pagila** sample database for PostgreSQL. The project demonstrates fundamental and advanced SQL techniques commonly used in data analysis and business intelligence.

---

## Project Overview

The objective of this project was to strengthen practical SQL skills by solving analytical tasks on a relational database.

The exercises cover:

- Data filtering and sorting
- Aggregation and grouping
- JOIN operations
- Subqueries
- Common Table Expressions (CTEs)
- Window functions
- Business-oriented SQL analysis

## Tasks Included

### Level 1 — Filtering and Selection

- Display films longer than 120 minutes
- List customers from a specific city
- Search films by title

### Level 2 — Aggregation and Grouping

- Count films by category
- Calculate total payments by customer
- Calculate average rental duration
- Filter categories using HAVING

### Level 3 — JOIN Operations

- Display films with their categories
- Retrieve rental history for a customer
- Calculate revenue by store
- Find the top 10 actors by number of films

### Level 4 — Window Functions

- Running total of customer payments
- Rank films by duration within each category
- Monthly revenue analysis with previous-month comparison
